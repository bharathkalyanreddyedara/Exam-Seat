Exam Seating Arrangement System - Login Demo

This demo shows the enhanced login functionality with separate admin and student login options. The actual implementation includes proper validation and dashboard redirection based on user role.

Login Interface

When clicking "Login" in the navbar, users are presented with these two options:

Login Options

Administrator Login Form

Default admin: username "admin", password "admin123"

Student Login Form

Default student: username "student1", password "student123"

Role-Specific Dashboards

After successful login, the user is redirected to the appropriate dashboard based on their role:

Admin Dashboard

Welcome back, admin!

This dashboard shows admin-specific features including:

  • Exam management options
  • Student management
  • Seating arrangement tools
  • Statistics and reports

Student Dashboard

Welcome back, student1!

This dashboard shows student-specific features including:

  • Upcoming exams list
  • Assigned seat information
  • Exam schedule viewer
  • Seating plan viewer

Additional Features

  • Role validation ensures students can't access admin features and vice versa
  • Clear error messages for incorrect credentials
  • Dashboard title and content dynamically updates based on user role
  • Proper session management with localStorage
  • Clean, responsive UI for all screen sizes
